System Design Architecture
Calloway Pharmacy Management Platform
Users

Admin

Manages users, permissions, settings, and backups.

Staff

Performs inventory, medicine lookup, and operations.

Cashier

Handles POS transactions and order fulfillment.

Customer

Places online orders and tracks order status.

Presentation Layer

Admin Dashboard UI

Dashboard, reports, system configuration, monitoring pages.

Staff and POS UI

Inventory management, POS, medicine-locator, notifications.

Customer Web Portal

Online ordering, order tracking, account pages.

Application Layer (PHP)

Routing and API Endpoints

PHP pages and JSON APIs receive requests and return responses.

Authentication and Authorization

Session management, role checks, permission checks, CSRF protection.

Business Modules

Orders, inventory, medicine locator, reports, settings, online ordering.

Notification and Activity Logging

POS notifications, operational alerts, and user activity logs.

Data and Operations Layer

MySQL or MariaDB Database

Users, roles, products, sales, online orders, notifications, logs, settings.

File Storage

Backup files, export files, and system log files.

Background Jobs and External Service

Scheduled jobs for backup and email, plus SMTP email delivery.
