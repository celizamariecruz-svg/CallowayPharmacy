üîÄ If Statements (Making Decisions)

The if statement lets your code make decisions. "If this is true, do this."

üí° Real World Analogy

Like a pharmacist checking prescriptions:

  • IF customer has valid ID ‚Üí sell the medicine
  • ELSE IF customer has expired ID ‚Üí ask for valid one
  • ELSE ‚Üí cannot proceed with sale

Basic If Statement

$stock = 50; if ($stock > 0) { echo "Product is available!"; } // Translation: // "IF stock is greater than 0, show the message"

If-Else Statement

$stock = 0; if ($stock > 0) { echo "In stock! You can buy."; } else { echo "Sorry, out of stock."; } // Translation: // "IF stock is greater than 0, say in stock // OTHERWISE (else), say out of stock"

If-Else If-Else (Multiple Conditions)

$stock = 15; if ($stock >= 100) { $status = "Well Stocked"; $color = "green"; } else if ($stock >= 20) { $status = "Normal Stock"; $color = "blue"; } else if ($stock > 0) { $status = "Low Stock - Reorder Soon!"; $color = "orange"; } else { $status = "OUT OF STOCK"; $color = "red"; } echo $status; // Output: "Low Stock - Reorder Soon!"
üí° How It Works

PHP checks conditions from top to bottom. Once it finds a TRUE condition, it runs that code and skips the rest. Like going through a checklist and stopping at the first match!

üîó Combining Conditions

Sometimes you need to check multiple things at once.

AND (&&) - Both Must Be True

$age = 25; $hasID = true; // Both conditions must be true if ($age >= 18 && $hasID) { echo "Can purchase restricted medicine"; } // Checks: // Is age >= 18? YES ‚úì // Has ID? YES ‚úì // Both true = Runs the code

OR (||) - At Least One Must Be True

$paymentMethod = "gcash"; // At least one condition must be true if ($paymentMethod === "cash" || $paymentMethod === "gcash") { echo "Payment accepted!"; } // Checks: // Is payment cash? NO ‚úó // Is payment gcash? YES ‚úì // At least one true = Runs the code

NOT (!) - Flip True/False

$isExpired = false; // The ! flips the value if (!$isExpired) { echo "Product is still good!"; } // Translation: // IF NOT expired (meaning: if fresh/valid) // Same as: if ($isExpired === false) { echo "Product is still good!"; }

üîÑ Loops (Repeating Actions)

Loops let you repeat code multiple times without writing it over and over.

üí° Real World Analogy

Like a pharmacist counting pills:

"Keep counting pills WHILE the count is less than 30"

or

"FOR each item in the order, add it to the receipt"

Foreach Loop (Most Common!)

Best for going through arrays. Used EVERYWHERE in your pharmacy system.

$medicines = ["Biogesic", "Neozep", "Bioflu"]; // Go through each item in the array foreach ($medicines as $medicine) { echo $medicine . "<br>"; } // Output: // Biogesic // Neozep // Bioflu // Translation: // "FOR EACH item in $medicines, call it $medicine and print it"

Foreach with Key-Value

$product = [ 'name' => "Biogesic", 'price' => 5.50, 'stock' => 100 ]; // Get both the key and value foreach ($product as $key => $value) { echo $key . ": " . $value . "<br>"; } // Output: // name: Biogesic // price: 5.50 // stock: 100

For Loop (When You Know The Count)

// Print numbers 1 to 5 for ($i = 1; $i <= 5; $i++) { echo "Count: " . $i . "<br>"; } // Output: // Count: 1 // Count: 2 // Count: 3 // Count: 4 // Count: 5 // The for loop has 3 parts: // $i = 1 ‚Üí Start at 1 // $i <= 5 ‚Üí Keep going while i is 5 or less // $i++ ‚Üí Add 1 after each loop

While Loop (Condition-Based)

$pills = 0; $prescription = 30; while ($pills < $prescription) { $pills++; // Add one pill echo "Added pill #" . $pills . "<br>"; } echo "Done! Total pills: " . $pills; // Translation: // "WHILE pills is less than prescription, keep adding pills"
‚ö†Ô∏è Infinite Loop Warning!

A while loop without a proper exit condition will run forever and crash your server!

// ‚ùå BAD - Infinite loop! while (true) { echo "Forever..."; // Never stops! } // ‚úì GOOD - Has exit condition while ($count < 10) { $count++; // Will eventually be >= 10 and stop }

üîß Functions (Reusable Code Blocks)

Functions are like recipes - write once, use many times!

üí° Real World Analogy

Think of functions like a coffee machine:

  • Input (Parameters): Coffee beans, water
  • Process (Function Body): Grind, brew, heat
  • Output (Return): Hot coffee

You don't rebuild the machine each time - you just use it!

Creating a Function

// Define a function function sayHello() { echo "Hello, welcome to Calloway Pharmacy!"; } // Call/use the function sayHello(); // Output: Hello, welcome to Calloway Pharmacy! sayHello(); // Can use it again!

Function with Parameters (Inputs)

// $name is a parameter - input that the function needs function greetCustomer($name) { echo "Hello, " . $name . "! Welcome!"; } // Call with different inputs greetCustomer("Maria"); // Output: Hello, Maria! Welcome! greetCustomer("Juan"); // Output: Hello, Juan! Welcome!

Function with Return Value (Output)

// Calculate total price with tax function calculateTotal($price, $quantity) { $subtotal = $price * $quantity; $tax = $subtotal * 0.12; // 12% VAT $total = $subtotal + $tax; return $total; // Send back the result } // Use the function $orderTotal = calculateTotal(100, 5); // Returns 560 echo "Your total is: ‚Ç±" . $orderTotal; // The function: // 1. Takes price (100) and quantity (5) // 2. Calculates subtotal: 100 √ó 5 = 500 // 3. Calculates tax: 500 √ó 0.12 = 60 // 4. Returns total: 500 + 60 = 560

Function with Default Parameters

// $discount has a default value of 0 function calculatePrice($price, $quantity, $discount = 0) { $subtotal = $price * $quantity; $discountAmount = $subtotal * ($discount / 100); return $subtotal - $discountAmount; } // Call without discount - uses default (0) echo calculatePrice(100, 2); // Output: 200 // Call with discount echo calculatePrice(100, 2, 10); // Output: 180 (10% off)

üìö Common Built-in Functions

PHP comes with thousands of ready-to-use functions. Here are the most common ones:

String Functions

$text = " Hello World "; strlen($text); // 17 - Get length trim($text); // "Hello World" - Remove spaces from ends strtoupper($text); // " HELLO WORLD " - Uppercase strtolower($text); // " hello world " - Lowercase str_replace("World", "PHP", $text); // " Hello PHP " - Replace substr($text, 2, 5); // "Hello" - Get part of string explode(" ", $text); // ["", "", "Hello", "World", "", ""] - Split to array

Array Functions

$items = ["apple", "banana", "cherry"]; count($items); // 3 - How many items in_array("banana", $items); // true - Check if exists array_push($items, "date"); // Add to end array_pop($items); // Remove from end, returns removed item sort($items); // Sort alphabetically array_reverse($items); // Reverse order implode(", ", $items); // "apple, banana, cherry" - Join to string

Number Functions

$price = 99.456; round($price); // 99 - Round to nearest integer round($price, 2); // 99.46 - Round to 2 decimal places floor($price); // 99 - Round down ceil($price); // 100 - Round up number_format($price, 2); // "99.46" - Format with commas and decimals $numbers = [10, 5, 20, 15]; max($numbers); // 20 - Highest value min($numbers); // 5 - Lowest value array_sum($numbers); // 50 - Add all together

Date/Time Functions

// Current date/time echo date("Y-m-d"); // "2025-12-19" echo date("H:i:s"); // "14:30:45" echo date("F j, Y"); // "December 19, 2025" // Common format codes: // Y = 4-digit year (2025) // m = Month number (01-12) // d = Day number (01-31) // H = Hour 24-format (00-23) // i = Minutes (00-59) // s = Seconds (00-59) // F = Full month name (December) // j = Day without leading zero (1-31) // Get current timestamp (seconds since 1970) $now = time(); // Convert string to timestamp $expiry = strtotime("2026-06-30");

üîÄ Switch Statement (Multiple Options)

When you have many possible values to check, switch is cleaner than many if-else.

$paymentMethod = "gcash"; switch ($paymentMethod) { case "cash": echo "Please pay at counter"; break; case "gcash": echo "Scan the GCash QR code"; break; case "maya": echo "Open your Maya app"; break; case "card": echo "Insert or tap your card"; break; default: echo "Unknown payment method"; } // Output: "Scan the GCash QR code"
‚ö†Ô∏è Don't Forget break!

Without break, PHP will continue running the next cases! This is a common bug.

‚ùì Ternary Operator (Short If-Else)

A one-line way to write simple if-else statements.

// Normal if-else if ($stock > 0) { $status = "Available"; } else { $status = "Out of Stock"; } // Same thing using ternary $status = ($stock > 0) ? "Available" : "Out of Stock"; // ‚Üë condition ‚Üë if true ‚Üë if false // Read as: // "Is stock > 0? If yes, use 'Available'. If no, use 'Out of Stock'."

More Examples

// Show discount badge $discount = 15; $badge = ($discount > 0) ? "$discount% OFF!" : ""; // Choose button color $isExpired = true; $buttonClass = $isExpired ? "btn-danger" : "btn-success"; // Default value (use ?? instead for null check) $name = $_GET['name'] ?? "Guest";
